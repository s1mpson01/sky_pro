# Проект на python

## Описание:

Это проект на бэкенде будет готовить данные 
для отображения в новом виджете.

## Установка:

1. Клонируйте репозиторий:
```
git clone https://github.com/s1mpson01/sky_pro.git
```
2. Установка окружения:
```
pip install poetry
poetry init
```
